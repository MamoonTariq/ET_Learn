### ET Learn

It is simple repo to learn NODE in MVC Patterns

**=> Setup:**
npm init

**=> Install express:**
npm install --save express

**=> Install Nodemon:**
npm i --save-dev nodemon

**=> Install Mongoose:**
npm i mongoose

**=> Install Env File:**
npm i dotenv

**=> Install bcrypt to convert password to hash:**
npm install bcrypt

**=> Install Json Web Token:**
npm install jsonwebtoken
